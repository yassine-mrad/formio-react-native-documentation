# Changelog

This page provides a detailed history of changes for each version of `formio-react-native`.

## Version 1.0.0 (2025-12-17)

### New Features

-   Initial release of `formio-react-native`.
-   Support for basic Formio components (Textfield, Email, Number, etc.).
-   Basic validation support.

### Enhancements

-   Improved performance for form rendering.

### Bug Fixes

-   Fixed an issue where form submission would not trigger correctly.
